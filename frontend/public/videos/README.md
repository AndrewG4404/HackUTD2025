# Hero Video Setup

## How to Add Your Video

1. **Place your video file(s) in this directory:**
   - Recommended: `hero-background.mp4` (MP4 format)
   - Optional: `hero-background.webm` (WebM format for better compression)

2. **Video Requirements:**
   - **Format**: MP4 (H.264 codec) or WebM
   - **Resolution**: 1920x1080 (Full HD) or higher
   - **Aspect Ratio**: 16:9 recommended
   - **Duration**: 10-30 seconds (will loop)
   - **File Size**: Keep under 10MB for fast loading
   - **Content**: Abstract, subtle background video that doesn't distract from text

3. **Video Optimization Tips:**
   - Use a video compression tool to reduce file size
   - Consider using WebM format for smaller file sizes
   - Keep the video subtle - it's a background element
   - Dark/abstract videos work best with the overlay

4. **If you don't have a video:**
   - The page will fall back to the gradient background
   - You can remove the `<video>` element from `app/page.tsx` if preferred

## Example Video Sources

- **Free stock videos**: Pexels, Pixabay, Unsplash
- **Abstract backgrounds**: Look for "abstract", "tech", "data visualization" themes
- **Bloomberg-style**: Professional, minimal, dark-themed videos
